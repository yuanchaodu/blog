# 第一个 Linux 命令

<img src="images/Linux.svg" width="300">

“第一个 Linux 命令”有几种理解方式：

1. **初学者通常学习的第一个命令**

   一般是 `ls`，用于查看当前目录中的文件和文件夹。

   ```bash
   ls
   ```

   示例输出：

   ```text
   Documents  Downloads  Pictures
   ```

2. **登录 Linux 后最值得先执行的命令**

   通常是 `pwd`，用于查看当前所在目录。

   ```bash
   pwd
   ```

   例如：

   ```text
   /home/nick
   ```

3. **从历史角度理解“第一个 Linux 命令”**

   Linux 并不存在官方认定的“第一个命令”。Linux 继承自 Unix，而早期 Unix 系统中最基础、最早出现的一批命令包括 `ls`、`cd`、`cp`、`mv` 等。

| 命令      | 作用       |
| ------- | -------- |
| `pwd`   | 查看当前目录   |
| `ls`    | 查看目录内容   |
| `cd`    | 切换目录     |
| `mkdir` | 创建目录     |
| `touch` | 创建文件     |
| `cp`    | 复制文件     |
| `mv`    | 移动或重命名文件 |
| `rm`    | 删除文件     |
| `cat`   | 查看文件内容   |

其中最经典、最常被认为是“第一个 Linux 命令”的是：

```bash
ls
```

因为它能立即让你看到系统里有什么内容。
