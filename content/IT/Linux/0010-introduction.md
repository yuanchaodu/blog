# Linux 简介

<img src="images/Linux.svg" width="300">


## 什么是 Linux

Linux 是一种开源的类 Unix 操作系统内核，由芬兰程序员 [Linus Torvalds](https://github.com/torvalds) 于 1991 年首次发布。基于 Linux 内核，全球开发者和组织构建了众多 Linux 发行版，如 Ubuntu、Debian、CentOS 和 Red Hat Enterprise Linux 等。

Linux 以稳定、安全、高性能和开放性著称，广泛应用于服务器、云计算平台、嵌入式设备、超级计算机以及个人电脑等领域。

## Linux 的特点

### 开源免费

Linux 遵循开源协议，用户可以自由获取、使用、修改和分发源代码。

### 稳定可靠

Linux 系统经过长期发展和大量实际应用验证，能够长时间稳定运行，特别适合服务器环境。

### 安全性高

Linux 具有完善的权限管理机制，同时拥有活跃的开源社区持续修复安全漏洞。

### 良好的可扩展性

Linux 支持丰富的软件和硬件平台，能够满足从个人学习到大型企业生产环境的各种需求。

### 强大的命令行工具

Linux 提供丰富的命令行工具，便于自动化运维、批量处理和系统管理。

## Linux 的应用场景

* Web 服务器
* 数据库服务器
* 云计算平台
* 容器与 Kubernetes 平台
* 网络设备
* 工业控制系统
* 物联网设备
* 超级计算机

目前全球大多数互联网服务和云计算基础设施都运行在 Linux 系统之上。

## 常见 Linux 发行版

| 发行版                      | 特点          | 适用场景     |
| ------------------------ | ----------- | -------- |
| Ubuntu                   | 易于使用，社区活跃   | 学习、开发、桌面 |
| Debian                   | 稳定可靠        | 服务器、开发   |
| Rocky Linux              | CentOS 替代方案 | 企业服务器    |
| Red Hat Enterprise Linux | 商业支持完善      | 企业生产环境   |
| openSUSE                 | 配置工具丰富      | 开发与测试    |

## Linux 文件系统结构

Linux 使用树状目录结构，所有文件和目录都从根目录 `/` 开始。

常见目录如下：

| 目录      | 作用        |
| ------- | --------- |
| `/`     | 根目录       |
| `/home` | 用户目录      |
| `/etc`  | 配置文件      |
| `/var`  | 日志和运行数据   |
| `/usr`  | 应用程序和共享资源 |
| `/bin`  | 基本命令      |
| `/tmp`  | 临时文件      |
| `/opt`  | 第三方软件     |

## 总结

Linux 是当前最重要的开源操作系统之一。凭借其稳定性、安全性和灵活性，Linux 已成为服务器、云计算和现代信息基础设施的重要基础。对于 IT 从业人员而言，掌握 Linux 基础知识和常用命令是非常重要的技能。
